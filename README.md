# 📚 Personality-Based Book Recommendation Web App

A modern and elegant frontend web application that recommends books based on your personality type.

This project is built using pure **HTML, CSS, and JavaScript** with a premium UI design and interactive features.

---

## 🌟 Features

- 🎭 Personality-based book recommendations
- 📖 Real book titles with short descriptions
- 📚 Fun facts about books on homepage
- 🎨 Premium UI with custom color palette
- 📱 Responsive layout
- ⚡ Dynamic DOM manipulation using JavaScript

---

## 🚀 How It Works

1. User selects a personality type from dropdown.
2. JavaScript captures the selected value.
3. Based on the value, book recommendations are dynamically injected into the DOM.
4. UI updates instantly without page reload.
